# dialog-base
Basic Dialog Class


## Installation

1. Include it in your project

   ```js
   import { Dialog } from "dialog.js" 
   ```

## Usage

   ```js

   const dialog = new Dialog()
   ```

## Options

   ```ts
   title: String;
   content: String;
   width: String;

   ```

## Methods

   ```ts
   dialog.remove()

   ```
